Ce module contient des outils Matlab sous forme de scripts
ou de fonctions. La plupart de ces fonctions n'ont aucune
dépendance ni entre elles, ni avec une autre fonction non
présente dans MatlabCommonTools.

Ces fonctions sont toutes documentées, et la documentation
peut être générée au format HTML en lançant le script
gendoc.m. Attention, ce script nécessite la toolbox M2HTML
disponible à l'adresse suivante :
        < http://www.artefact.tk/software/matlab/m2html/ >
La documentation ainsi créée se trouve alors dans le sous
répertoire ./doc qu'il faut exclure du CVS.
